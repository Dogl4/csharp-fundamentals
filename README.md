## Run

To run use these commands in the terminal:

```bash
# download
git clone https://github.com/Dogl4/csharp-fundamentals.git

cd csharp-fundamentals

# selected project
cd /Caculator/
# or
cd /StopWatch/
# or
cd /TextEditor/
#or
cd /EditorHtml/
#or
cd /MyTodo/

# install dependencies
dotnet restore

# run application
dotnet run
```
