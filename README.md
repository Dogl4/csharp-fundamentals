# C# fundamentals

- Stopwatch
- Calculator
- TextEditor

To run use comand in terminal:
```bash
# download
git clone https://github.com/Dogl4/csharp-fundamentals.git

# selected project
cd /Caculator/
# or
cd /StopWatch/
# or
cd /TextEditor/

# install dependencies
dotnet restore

# run application
dotnet run
```
